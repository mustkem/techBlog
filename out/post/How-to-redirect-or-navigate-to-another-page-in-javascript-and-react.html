How to redirect or navigate to another page in javascript and react

Changing route in javascript and react is handled differently. In javascript, it is straightforward. javascript has some methods on the window object. We can use these methods and change location easily.

Change location using JavaScript

To change to a location use location.href.

window.location.href='https://otherpage.com';

This will change location to https://otherpage.com.

Replace location using JavaScript

 window.location.replace('https://anotherpage.com');

This will replace location to https://another.com.


Routing in React

In React, routing is done by using third-party packages for example react-router-dom, react-router-redux, wouter, etc.  These packages use history API internally for routing.

Take an example of react-router-dom. 

Change location

history.push(location)

This will change location and your componentDidMount() method will be invoked so that HTTP requests can be made.

Replace location
In the same way, you can replace location using replace method.

history.replace(location)


React Router uses the history API of javascript under the hood and exposes a wrapper for navigation to the target locations. This wrapper lets you easily manage session history anywhere JavaScript runs. A history object abstracts away the differences in various environments and provides a minimal API that lets you navigate, manage the history stack, and persist state between sessions.
